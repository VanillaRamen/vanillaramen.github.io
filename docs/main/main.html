About Me

Hello! My real name isn't important, but online I go by "Vani."

I'm a first-year computer engineering major trying to find happiness through the things I like.

Even though computer engineering involves both hardware and software, most of my experience so far has involved software. I'm most familiar with JavaScript, but I'm building experience with Python and C++ at the moment. I'm not an expert in JS by any means, though, and especially not in the context of web development (haha)!

If you want to see what I've done so far, check out my GitHub! This website is hosted there, and my other projects can be found there too!

There are only small projects there for now, but eventually, I'll make and post more ambitious projects.
