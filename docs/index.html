    About Me

    Aspirations

    This is an unreasonable goal, but I think one day I would love to have a keyboard that I made entirely from scratch, down to the computer chip handling all of the logic and protocols (except connectivity like Bluetooth and USB).

    I know I could just use an existing programmable little computer thing like an Arduino or an ESP32, but wouldn't it be so cool to say that I designed a little chip myself? And it's not like I would need that much computational power.
